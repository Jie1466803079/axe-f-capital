# AXE.F Capital Pty Ltd — Grocery store marketing site

A single-page marketing website for AXE.F Capital Pty Ltd (grocery retailer). Built with React and Vite.

- Email: fengwenye00@gmail.com
- ABN: 22 639 805 511

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
```

Output is in `dist/`. Preview with:

```bash
npm run preview
```

## What’s included

- **Sections:** Hero, About, Offerings, Visit, Contact
- **Footer:** Address, email, ABN, nav links
- **Contact form:** Opens your email client with a pre-filled message to the store
- **Get directions:** Links to Google Maps for 65 John Street, Cabramatta NSW 2166
- **Responsive:** Mobile-first layout; nav collapses to a menu on small screens

No backend, cart, or checkout — this is a marketing/landing site only.
